# TechArmyKnife

A developer toolkit with 18 tools for encoding, formatting, diffing, converting, and generating.

Built with React, Vite, and a git-inspired dark UI. Everything runs 100% client-side. No server-side processing, so no data leaves your browser.

Vibe coded with Claude.

## Tools

**API Tools**
- **cURL Generator** — Generate cURL commands
- **k6 Generator** — Generate k6 load test scripts from cURL commands

**Formatting & Viewers**
- **JSON Diff** — Git-style side-by-side JSON payload diff with smart comparison options
- **JSON Viewer** — Format, beautify and view JSON data
- **YAML Viewer** — Format, beautify and view YAML data
- **XML Viewer** — View and analyze XML data
- **SQL Formatter** — Format SQL queries

**Encoding & Conversion**
- **Base64 Converter** — Encode or decode Base64 strings
- **URL Encoder** — Encode or decode URL strings
- **JWT Tool** — Encode and decode JWT tokens
- **Hash Generator** — Generate MD5, SHA1, SHA256, SHA512 hashes
- **JSON ⟷ YAML** — Convert between JSON and YAML
- **JSON ⟷ XML** — Convert between JSON and XML
- **CSV ⟷ JSON** — Convert between CSV and JSON

**Generators & Utilities**
- **UUID Generator** — Generate and validate UUIDs
- **UNIX Timestamp** — Convert Unix timestamps and dates
- **Cron Calculator** — Create and analyze cron expressions

**Text & Content**
- **Regex Tester** — Test and generate regex patterns

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build and dev server
- **Tailwind CSS** — utility styling

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
